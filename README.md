# GitHub User Finder – Поиск пользователей GitHub

**Автор:** Таулан Сарбашев

## Описание
Приложение с графическим интерфейсом (Tkinter) для поиска пользователей GitHub через официальное GitHub REST API.  
Возможности:
- Поиск пользователей по имени (логину) – поле ввода, кнопка «Поиск»
- Результаты отображаются в виде списка с отметкой ★ для избранных
- Добавление пользователей в **избранное** (сохраняется в JSON)
- Удаление из избранного
- Открытие профиля пользователя в браузере (как из результатов, так и из избранного)
- Автоматическая загрузка и сохранение избранного в `favorites.json`
- Проверка: поле поиска не может быть пустым

## Как использовать API
Приложение использует публичный GitHub API без аутентификации:
- Эндпоинт: `https://api.github.com/search/users?q={query}&per_page=20`
- Лимит: для неавторизованных запросов — 60 запросов в час (с одного IP). Для тестирования этого достаточно.
- При желании можно добавить токен (изменив `headers` в `requests.get`), но в данном проекте он не требуется.
